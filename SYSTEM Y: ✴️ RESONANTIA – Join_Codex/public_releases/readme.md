## 🜇 Overview · Überblick  

This directory contains all official **press releases, statements, and public resources** of the **Scarabæus1033 / NEXAH-CODEX** initiative.

## 🎯 Purpose · Zielsetzung  

The **Public Releases Section** exists to ensure:  
Diese **Public-Releases-Sektion** stellt sicher:  

- transparent communication with media and research partners  
- open access to verified texts, images, and data  
- documentation of artistic-scientific collaboration  
- clarity on authorship, licensing, and attribution  

- transparente Kommunikation mit Medien und Forschungspartnern  
- offener Zugang zu geprüften Texten, Bildern und Daten  
- Dokumentation der künstlerisch-wissenschaftlichen Zusammenarbeit  
- Klarheit über Autorschaft, Lizenzierung und Zitation
