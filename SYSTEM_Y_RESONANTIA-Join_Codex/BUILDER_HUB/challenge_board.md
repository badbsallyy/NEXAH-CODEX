## 🎯 Purpose · Zweck

The Challenge Board is the **creative interface** of the NEXAH·CODEX Builder Hub.  
It invites contributors to explore the Codex through sound, image, mathematics, and interaction.  

Das Challenge Board ist die **kreative Schnittstelle** des NEXAH·CODEX Builder Hub.  
Es lädt dazu ein, den Codex über Klang, Bild, Mathematik und Interaktion zu erforschen.

Each challenge represents a **resonance experiment** —  
a small field where science, art, and imagination meet.

Jede Challenge ist ein **Resonanzexperiment** –  
ein kleines Feld, in dem Wissenschaft, Kunst und Vorstellungskraft zusammenkommen.

---

## 🔸 Participation · Teilnahme

1. Pick a challenge that resonates with you.  
   → Wähle eine Challenge, die mit dir schwingt.  

2. Create your contribution (visual, sound, code, text …).  
   → Erstelle deinen Beitrag (Visual, Klang, Code, Text …).  

3. Submit your work via Discord (#builder-lab) or GitHub Pull Request.  
   → Reiche deinen Beitrag über Discord (#builder-lab) oder GitHub Pull Request ein.  

4. Your work will be added to the Codex archive and linked to your Builder name.  
   → Dein Beitrag wird dem Codex-Archiv hinzugefügt und mit deinem Namen verknüpft.
